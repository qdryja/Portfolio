Bachelor Project 2025

Data-Driven Prediction of ATP Tennis Match Outcomes Using Machine Learning Techniques

This is my most recent and most significant project, which I’m very proud of. I focused on predicting ATP tennis match outcomes using advanced machine learning techniques. The work involved collecting and cleaning decades of match data, engineering performance-driven features, and experimenting with multiple predictive models. Following extensive fine-tuning and validation, the model not only achieved high predictive accuracy on Grand Slam tournaments between 2010–2024 but also, when combined with strategic betting approaches, consistently outperformed bookmaker odds. This project not only deepened my expertise in data science and model optimization but also strengthened my ability to connect technical results with real-world applications.
